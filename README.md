 🎬 Predictive Movie Recommendation & Streaming Service

A modular Java-based streaming web application delivering 
personalized movie recommendations through intelligent 
filtering algorithms and clean layered architecture.

 🛠️ Tech Stack
- **Backend:** Java, Spring Boot, Spring MVC, REST APIs
- **Frontend:** Bootstrap 5, jQuery, AJAX, Thymeleaf
- **Database:** MySQL
- **Testing:** JUnit 5
- **Tools:** Git, Maven, Postman

 ✨ Features
- Real-time movie recommendation engine using Java 
  Streams API and Collections framework
- Secure RESTful endpoints for recommendation data delivery
- Asynchronous UI updates via jQuery — zero page reloads
- 95% JUnit test coverage on all recommendation workflows
- Clean MVC architecture with strict separation of concerns

 📊 Performance
- Optimized recommendation sorting with optimal 
  time complexity
- Zero page reload architecture for instant user experience

🚀 Architecture
Built on Spring MVC enforcing strict separation between:
- Data access layer (Repository)
- Business logic layer (Service)
- Presentation layer (Controller/View)
